IPAM steht für IP Address Management. Dies ist eine Methode zur Planung, Verwaltung und Überwachung von IP-Adressbereichen in Netzwerken. 
Somit wird es ermöglicht die IP-Adressen automatisch zuzuweisen und integriert DNS- und DHCP-Server. 

Schließlich reduziert man die manuellen Fehler und ersetzt [[Spreadsheets]].

#### **Beispiele IPAM-Tools:** 
1. IP-Adressverwaltung von Windows Server
2. Open-Source-Lösung phpIPAM
3. Solarwinds IP Address Manager und der kostenlose IP Address Tracker 
4. ManageEngine OpenUtils
5. Infoblox BloxOne DDI und NIOS DDI
6. Integrity von Bluebox
7. Tidal Lightmesh
