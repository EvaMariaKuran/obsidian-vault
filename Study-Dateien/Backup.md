Ein [[Backup]] ist eine [[Sicherheitskopie]], die automatisch durch eine [[Software]] oder manuell durch einen [[Anwender]] erstellt wird. 
Das Duplikat umfasst einen [[Datenbestände]], der regelmäßig auf dem [[Computer]], auf einer externen [[Festplatte]], in einer [[Cloud]] oder auf einem einfachen [[USB-Stick]] gespeichert wird. 

Ein [[Backup]] führt man regelmäßig durch, um [[Datenverlust]] vorzubeugen, wenn ein [[Hardware-Ausfall]], [[Viren]], [[Diebstahl]] oder auch wenn versehentlich eine [[Löschung]] durchgeführt wurde oder passiert ist. 
Das [[Ziel]] einer [[Datensicherung]] ist es, dass der [[Datenverlust]] so gering wie möglich bis gar nicht besteht. 

Bei der Durchführung eines [[Backup]]s gibt es auch verschiedene [[Arten_Backup]].
