Die verbindungssuchenden [[Geräte]] fordern die [[IP-Adresskonfiguration]] von einem [[DHCP-Server]] an, der seinerseits auf eine [[Datenbank]] zurückgreift, in der die einzustellenden [[Netzwerkparameter]] vorgegeben sind.
Unter anderem kann dieser [[Server]], der [[Bestandteile]] jedes modernen [[DSL-Router]], dem [[Client]] mithilfe seiner [[Datenbankinformation]]en folgende [[Einstellungen]] zuweisen: 
	- eindeutige [[IP-Adressen]]
	- [[Subnetzmaske]]
	- Standard-[[Gateways]]
	- [[DNS-Server]]
	- [[Proxy-Konfiguration]] via [[WPAD]] ([[Web Proxy Auto-Discovery Protocol]])
