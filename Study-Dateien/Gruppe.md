Eine [[Gruppe]] bezeichnet einen [[Zusammenschluss]] von mindestens drei [[Personen]], [[Geräte]], Dingen oder ähnliches ([[Gruppenmitglieder]]), die über einen längeren Zeitraum ein gemeinsames [[Ziel]] verfolgen, in einem andauernden Austausch stehen und dabei ein [[Zusammengehörigkeitsgefühl]] empfinden.


Definition 1.1 (**Gruppen**). (a) Eine **Gruppe** ist eine Menge G zusammen mit einer „Verknüpfung“, d. h. einer Abbildung · : G×G → G (a,b) ↦→ a·b so dass die folgenden Eigenschaften (auch Gruppenaxiome genannt) gelten: (G1) Für alle a,b,c ∈ G gilt (a·b)·c = a·(b·c) (Assoziativität).
