Eine [[Gruppenrichtlinie]] ist eine [[Windows-Funktion]] bzw. eine von vielen [[AD-Funktionalitäten]], die eine Vielzahl von erweiterten [[Einstellungen]] Ihres [[Betriebssysteme]] enthält. 
Insbesondere für [[Netzwerkadministratoren]], [[lokale Gruppenrichtlinien]] können jedoch auch verwendet werden, um [[Einstellungen]] auf einem einzelnen [[Computer]] anzupassen. 

[[Gruppenrichtlinie]]n lassen sich im [[Active Directory]] als [[AD-Objekten]] speichern.
