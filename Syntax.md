Die [[Syntax]] von einer von den vielen [[Programmiersprachen]] beschreibt, welche [[Kombination]] von [[Zeichen]] fehlerfreie [[Programm]]e der Sprache bilden. 
So meint man in der [[Informatik]] die [[Regeln]], die [[Programmiersprachen]] beim Schreiben von [[Quellcode]] vorgeben. 
Sodass jede [[Programmiersprachen]] über eine eigene [[Syntax]] verfügt, die für Befehle innerhalb des [[Zeichencodes]] befolgt werden muss. 

Bei der [[Programmierung]] von [[MyKa]] wird die [[Syntax]] durch die [[Menge]] der möglichen [[MyKa-Programme]] festgelegt. 
Eine [[Präzisierung]] der [[Menge]] kann z.B. mit Hilfe einer [[Grammatik]] vorgenommen werden.
