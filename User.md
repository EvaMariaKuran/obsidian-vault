Das Wort [[User]] leitet sich vom englischen Wort "use" ab ("benutzen" oder "verwenden"). 
Der [[Fachbegriff]] bezeichnet üblicherweise eine [[Personen]], die ein bestimmtes [[System]], eine [[Software]], eine [[Plattform]] oder einen [[Dienste]] nutzt. 
Im technologischen Kontext wird "[[User]]" oft als [[Synonym]] für "[[Benutzer]]" verwendet.
